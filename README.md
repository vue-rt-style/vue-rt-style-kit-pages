# vue-rt-style-kit-pages

[DEMO-стенд](https://vue-rt-style.github.io/vue-rt-style-kit-pages/docs/)

Для работы потребуются симлинки. Чтобы их создать запустите:
````
./create-symlinks.sh
````

Простой запуск проекта:

````
npm start
````
